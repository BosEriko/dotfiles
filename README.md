# Dotfiles

My configuration:

- Navigate TMUX panes using alt + h, j, k or l

## Installation

    rm -rf .vim
    rm .vimrc
    rm .tmux.conf

    git clone git@github.com:BosEriko/dotfiles.git
    ln -s dotfiles/vimrc ~/.vimrc
    ln -s dotfiles/tmux.conf ~/.tmux.conf
